# 4단계 Arduino 코드

[`04_modular_controller.ino`](04_modular_controller.ino)는 센서 읽기, 상태 결정, 네오픽셀 출력, 시리얼 출력을 기능별 함수로 나눈 통합 코드입니다.

## 현재 적용된 기능

- 가변저항: 네오픽셀 밝기 `0~80` 조절
- 터치센서: 누를 때마다 파랑·주황·보라·초록 순서로 색 변경
- 조도센서: 원시값 측정 및 시리얼 출력
- 초음파센서: 거리 측정 및 시리얼 출력
- 네오픽셀: 결정된 색과 밝기 출력

조도와 거리를 조명 자동 제어에 반영하는 기준은 실제 키트의 측정값을 확인한 뒤 추가합니다.
