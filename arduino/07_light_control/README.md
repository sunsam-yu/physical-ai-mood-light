# 7단계 Arduino 조명 명령 수신

웹에서 보낸 `LIGHT,R,G,B,밝기,모드` 한 줄을 검사한 뒤 네오픽셀에 적용합니다.

## 확인 순서

1. `Adafruit NeoPixel` 라이브러리를 설치합니다.
2. `07_light_control.ino`를 UNO에 업로드합니다.
3. 시리얼 모니터를 닫고 7단계 웹페이지에서 Web Serial을 연결합니다.
4. 색과 밝기를 보냅니다.
5. 실제 LED와 2D 미리보기를 비교합니다.

잘못된 필드 수·숫자·범위·모드는 LED에 적용하지 않습니다. 정상 적용 후에는 `light_state` JSON을 돌려줍니다. 7단계 웹은 쓰기 흐름에 집중하므로 이 응답을 화면에 표시하지 않으며, 9단계 통합본에서 실제 상태 동기화에 사용합니다.
