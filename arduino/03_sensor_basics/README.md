# 3단계 Arduino 코드

센서와 네오픽셀을 하나씩 실행하면서 입력과 출력의 의미를 확인하는 예제입니다.

| 순서 | 예제 | 확인할 내용 |
|---:|---|---|
| 1 | [`01_potentiometer`](01_potentiometer) | 가변저항의 위치가 `0~1023` 값으로 바뀌는 과정 |
| 2 | [`02_light_sensor`](02_light_sensor) | 빛의 세기에 따른 아날로그 값 변화 |
| 3 | [`03_touch_sensor`](03_touch_sensor) | 터치 여부가 `HIGH·LOW`로 바뀌는 과정 |
| 4 | [`04_ultrasonic_sensor`](04_ultrasonic_sensor) | 왕복 시간을 거리로 환산하는 과정 |
| 5 | [`05_neopixel`](05_neopixel) | RGB 값으로 색을 만드는 과정 |

각 폴더에서 폴더와 이름이 같은 `.ino` 파일을 Arduino IDE로 엽니다. 시리얼 모니터는 모두 `115200 baud`를 사용합니다.

## 아직 확정하지 않은 값

조도센서는 키트의 회로 구성에 따라 밝을 때 값이 커질 수도, 작아질 수도 있습니다. 실제 키트로 측정하기 전에는 방향과 기준값을 코드에 고정하지 않습니다.
