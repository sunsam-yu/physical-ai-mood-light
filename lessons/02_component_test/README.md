# 2단계. 키트 조립 및 부품 점검

[이전 단계: 프로젝트 이해](../01_project_overview/README.md) · [프로젝트 전체 보기](../../README.md)

## 학습목표

- 키트의 센서와 네오픽셀 핀을 확인할 수 있다.
- 시리얼 모니터에서 네 종류의 입력값을 확인할 수 있다.
- 센서 입력에 따라 네오픽셀의 밝기와 색이 달라지는지 점검할 수 있다.

## 완성 결과

프로그램을 실행하면 시리얼 모니터에 가변저항, 조도, 터치, 거리값이 한 줄로 반복 표시됩니다. 가변저항을 돌리면 LED 밝기가 달라지고, 터치센서를 누르면 LED가 파란색에서 주황색으로 바뀝니다.

## 준비물

- Arduino UNO와 USB 케이블
- 키트에 포함된 센서 보드와 네오픽셀 8구
- Arduino IDE가 설치된 컴퓨터
- `Adafruit NeoPixel` 라이브러리

## 핀 확인

| 부품 | Arduino UNO 핀 |
|---|---|
| 가변저항 | A0 |
| 조도센서 | A5 |
| 터치센서 | D13 |
| 초음파센서 TRIG | D2 |
| 초음파센서 ECHO | D3 |
| 네오픽셀 | D7 |

전원을 연결하기 전에 매뉴얼의 결선과 위 핀 번호가 일치하는지 확인합니다.

## 실행 순서

1. Arduino IDE에서 `02_component_test.ino`를 엽니다.
2. 라이브러리 관리자에서 `Adafruit NeoPixel`을 설치합니다.
3. 보드를 `Arduino Uno`, 포트를 연결된 포트로 선택합니다.
4. 코드를 업로드합니다.
5. 시리얼 모니터를 열고 속도를 `115200 baud`로 맞춥니다.
6. 각 센서를 한 번씩 조작하며 값의 변화를 확인합니다.

## 작동 확인표

- [ ] 가변저항을 돌리면 `0~1023` 범위의 값이 변한다.
- [ ] 조도센서를 가리거나 빛을 비추면 값이 변한다.
- [ ] 터치센서를 누르면 `없음`이 `감지`로 변한다.
- [ ] 초음파센서 앞 물체를 움직이면 거리값이 변한다.
- [ ] 가변저항을 돌리면 네오픽셀 밝기가 변한다.
- [ ] 터치센서를 누르면 네오픽셀 색이 바뀐다.

## 학생이 수정할 부분

코드 맨 위의 `학생이 확인하거나 바꿀 설정`만 수정합니다. 처음에는 핀 번호를 바꾸지 말고 `PRINT_INTERVAL_MS`를 `100`, `500`, `1000`으로 바꾸며 출력 주기의 차이를 확인합니다.

## 오류 해결

| 증상 | 먼저 확인할 것 |
|---|---|
| 업로드가 되지 않음 | 보드와 포트 선택, USB 케이블 연결 |
| 글자가 깨짐 | 시리얼 모니터 속도를 115200으로 설정 |
| 네오픽셀이 켜지지 않음 | D7 연결, 5V와 GND 방향, 라이브러리 설치 |
| 거리값이 계속 측정 실패 | D2·D3 연결과 TRIG·ECHO 방향 |
| 센서값이 변하지 않음 | 해당 센서의 전원·GND·신호 핀 |
