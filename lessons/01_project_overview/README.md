# 1단계. 프로젝트 이해와 완성 모습

[프로젝트 전체 보기](../../README.md) · [다음 단계: 키트 조립 및 부품 점검](../02_component_test/README.md)

## 프로젝트 질문

사람의 표정과 주변 상황을 감지해 실제 조명과 화면 속 조명이 함께 반응하도록 만들 수 있을까?

## 우리가 만들 작품

웹캠이 얼굴 표정 특징을 분석하면 웹 대시보드가 표정 분류 결과와 센서값을 함께 보여 줍니다. 결정된 조명색은 실제 네오픽셀과 화면 속 가상 무드등에 동시에 적용됩니다.

> 이 프로젝트는 사람의 진짜 감정을 판정하지 않습니다. 웹캠에 나타난 얼굴의 표정 특징을 몇 가지 범주로 분류합니다.

## 부품별 역할

| 부품 | 작품에서 하는 일 |
|---|---|
| 웹캠 | 얼굴 영상 입력 |
| 표정 분류 AI | 기쁨·중립·놀람·찡그림 중 하나를 추정 |
| 초음파센서 | 사람이 가까이 왔는지 확인 |
| 조도센서 | 주변 밝기에 맞게 LED 밝기 보정 |
| 가변저항 | 사용자가 최대 밝기 조절 |
| 터치센서 | 자동·수동 모드 전환 |
| 네오픽셀 | 결정된 색과 밝기를 실제로 출력 |
| 웹 대시보드 | 센서값, AI 결과, 조명 상태 표시 및 제어 |

## 전체 작동 순서

1. 초음파센서가 사람의 접근을 확인합니다.
2. 웹캠 영상에서 얼굴 표정 특징을 찾습니다.
3. AI가 표정 범주와 확률을 제시합니다.
4. 주변 조도와 사용자가 정한 최대 밝기를 반영합니다.
5. 실제 네오픽셀과 화면 속 무드등이 같은 상태로 바뀝니다.
6. 사용자는 웹에서 조명을 직접 바꾸거나 자동 모드로 돌아갈 수 있습니다.

## 수업에서 지킬 원칙

- 각 단계가 정상 작동한 뒤 다음 단계로 넘어갑니다.
- 코드에서는 표시된 설정값과 지정된 함수만 수정합니다.
- 얼굴 사진과 웹캠 영상은 저장하지 않습니다.
- AI 결과는 감정 판정이 아니라 표정 특징에 따른 추정값으로 설명합니다.

## 시작 전 확인

- [ ] 최종 작품의 입력, 처리, 출력을 말로 설명할 수 있다.
- [ ] 센서마다 맡은 역할을 한 가지씩 설명할 수 있다.
- [ ] 표정 분류 결과가 실제 감정과 다를 수 있음을 이해한다.
